# IT0079 KOTLIN: MIDTERM EXAM REVIEWER

A GitHub Pages reviewer for IT0079 Kotlin Formative Assessments 1, 2, 3, and 4.

## Included question sets
- Formative 1: 23 text-only questions
- Formative 2: 53 text-only questions
- Formative 3: 40 questions
- Formative 4: 46 questions
- Total: 162 questions

Questions from Formative 1 and Formative 2 that require screenshots or images were intentionally excluded.

## Features
- Wrong answers turn red
- Correct answers turn green
- Continue trying until the correct answer is chosen
- Previous and Next buttons
- Clickable question-number grid
- Separate FA1, FA2, FA3, FA4, and All Questions modes
- Shuffle and reset buttons
- Local progress saving

## Upload to GitHub Pages
1. Replace the existing `index.html` and `README.md` in your repository with the updated files.
2. Commit the changes.
3. GitHub Pages will redeploy automatically.
